About

The Thames River Sentiment Dashboard is a collaboration between the University of Oxford and Thames21. This project is supported by the University of Oxford's ESRC Impact Acceleration Account.

The dashboard is built with emotional AI, natural language processing, and data from the Twitter API and the Environment Agency. Please visit our GitHub to learn more about our methods and data.

For feedback please send us an email or follow us on Twitter.
